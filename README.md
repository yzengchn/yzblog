# yzblog

该项目是一个典型的由springcloud管理的微服务项目，主要包括如下模块

|     服务名     | 父级依赖 | 模块说明 |
| :------------: | :------: | :------: |
| yzblog_parent  |    无    |  父工程  |
| yzblog_common  |  parent  |  通用包  |
|  yzblog_base   |  parent  | 基础服务 |
| yzblog_article |  parent  | 文章服务 |
|  yzblog_faqs   |  parent  | 问答服务 |
| yzblog_search  |  parent  | 搜索服务 |
|   yzblog_sms   |  parent  | 短信服务 |
|                |          |          |
